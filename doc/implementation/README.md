## Как пользоваться планом

1. Откройте [IMPLEMENTATION.md](./IMPLEMENTATION.md)
2. Работайте по фазам — каждая фаза самодостаточна и тестируема
3. Отмечайте выполненные задачи: `- [ ]` → `- [x]`
4. Проверяйте **Acceptance Criteria** каждой задачи перед переходом к следующей
5. Не переходите к фазе, пока не завершены её зависимости
